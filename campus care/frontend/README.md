# Campus Care — Frontend (minimal)

This is a minimal frontend scaffold for the Campus Care app. It uses React from CDNs and Babel standalone for quick dev/demo without a bundler.

How to run:

1. Open `index.html` in a browser (double-click or use a local static server).

Notes:
- This is intentionally a simple in-browser setup for frontend work only.
- You can later convert this to a Vite/CRA project when you add a backend and build tooling.
